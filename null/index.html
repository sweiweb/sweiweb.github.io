windows子系统WSL的配置
发表于 2022-01-11| 更新于 2022-01-11|WSL
阅读量:|评论数:

Windows Subsystem for Linux（简称WSL）是一个在Windows 10上能够运行原生Linux二进制可执行文件（ELF格式）的兼容层。它是由微软与Canonical公司合作开发，其目标是使纯正的Ubuntu、Debian等映像能下载和解压到用户的本地计算机，并且映像内的工具和实用工具能在此子系统上原生运行。

开启WSL

  • 打开设置

  • 进入应用

  • 选择应用和功能

  • 点击程序和功能

  • 点击启用或关闭Windows功能

  • 勾选适用于Linux的Windows子系统

安装Linux发行版

打开Win10应用商店搜索你喜欢的Linux发行版并安装，如Ubuntu，Kali Linux，Debian等

这里我们以Ubuntu为例继续。

一般情况下，程序包内还要下载Linux软件包，占用空间可能很大。我们可以将下载的.appx程序修改扩展名为.zip，解压到D盘后，开打解压文件夹下的ubuntu.exe进行安装。

安装完成后按提示输入默认用户名、密码。

安装 termux 后，可以安装 hexo，在手机上搭建博客。

安装 hexo

输入以下命令即可

初始化 hexo

此时，可在本机浏览器中输入127.0.0.1:4000查看效果

如果想在其他浏览器中搜索到自己的博客，可以在 github 上托管博客

将博客托管在github平台

创建仓库

首先登入 github，在右上角+点击 New repository 创建一个仓库，在 repository name 中输入 github账户名.github.io ，同时勾选下方的 Initialize this repository with a README。

连接 hexo 和 git

在终端输入命令设置用户名称和邮箱地址

使用 ssh-keygen 生成私钥和公钥

注意查看输出信息的私钥id_rsa.pub所在地址,可以用命令cat id_rsa.pub或vi id_rsa.pub 查看私钥内容。然后登录 Github，点击右上角头像下的settings，点击 SSH and GPG keys 菜单下的 New SSH key 添加 ssh，将 id_rsa.pub 文件里的内容复制上去。随后，可以在终端输入命令 ssh -T git@github.com 测试添加 ssh 是否成功。

配置Deployment

在 blog 目录下对文件 _config.yml 进行编辑，输入 vi _config.yml，修改文件末尾的 Deployment

其中的 repository 值可以在 github 账户下 your repositories 的 clone or downlord 中，选择 Ues SSH 查看。修改完成后输入 :wq 保存并退出

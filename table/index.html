数据的表格可视化
发表于 2021-06-16| 更新于 2022-01-10|linux
阅读量:|评论数:

脚本目的： 将文件内数据按表格样式在终端显示，并且可以实现简单的数值运算操作。

代码

说明

用法：终端输入./table file 即可

- 表示此位置为空

--- 或者三个以上的-表示加入一个水平线

将 -[0~5] 输入到 ./table file 后面可以改变线框类型

=$1+$3 表示将此列的第一个数据和第三个数据进行数值计算

>=cos($2)+1 表示将次行的第二个数值参与到数值计算中

注意：数值计算仅按照从左到右，从上到下的优先级顺序进行计算。
